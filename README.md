# XP Inc. - Full Stack Developer Bootcamp

Bem-vindo ao repositório do Bootcamp Full Stack Developer da XP Inc.! Este bootcamp é projetado para ensinar e capacitar desenvolvedores com as habilidades necessárias para se tornarem desenvolvedores full stack proficientes, utilizando as tecnologias mais demandadas no mercado.

## Tecnologias Utilizadas

Durante o bootcamp, vamos explorar e utilizar as seguintes tecnologias:

- **C#**
- **.NET**
- **React**
- **Cloud Azure**
- **Banco de Dados**
- **API**
- **TypeScript**

## Estrutura do Projeto

O bootcamp será dividido em várias etapas, cobrindo cada uma das tecnologias mencionadas. Cada etapa terá uma série de projetos práticos e exercícios para ajudar a reforçar o aprendizado.

### 1. Introdução ao C# e .NET
- Fundamentos do C#
- Estrutura e funcionamento do .NET
- Desenvolvimento de aplicações básicas

### 2. Desenvolvimento Frontend com React
- Fundamentos do React
- Componentes, estado e props
- Hooks e gerenciamento de estado

### 3. Banco de Dados
- Introdução a bancos de dados relacionais e não relacionais
- Criação e gerenciamento de bases de dados
- Conexão com aplicações .NET

### 4. Criação de APIs
- Fundamentos de APIs RESTful
- Criação e consumo de APIs com .NET
- Autenticação e autorização

### 5. TypeScript
- Fundamentos do TypeScript
- Tipagem estática e interfaces
- Uso do TypeScript com React

### 6. Cloud Azure
- Introdução ao Azure
- Serviços básicos do Azure
- Deploy de aplicações no Azure

## Como Contribuir

1. Faça um fork deste repositório.
2. Crie uma branch para a sua feature (`git checkout -b feature/nova-feature`).
3. Commit suas mudanças (`git commit -m 'Adiciona nova feature'`).
4. Faça o push para a branch (`git push origin feature/nova-feature`).
5. Abra um Pull Request.
